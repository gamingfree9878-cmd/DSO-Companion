# DSO Companion 1.3.2

## Edelsteine
- Automatische Übernahme nur noch von einer hohen Stufe auf alle niedrigeren Stufen.
- Beispiel: Imperial Exquisit 10 → alle vorherigen Stufen ebenfalls 10.
- Trapez verändert keine höhere Stufe.
- Plus/Minus verändert weiterhin nur die ausgewählte Stufe.
- Neue Übersicht oben mit kleinem Edelsteinbild, Name und Anzahl der maximalen Steine.

## Runen
- Rune der Tapferkeit ergänzt.
- Kategorie: Defensiv.
- Bild aus dem hochgeladenen Screenshot.
- Verwendet die defensiven Staub- und Goldwerte.
